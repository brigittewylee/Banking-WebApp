PreRequisites
=
- Node.js v24.13.0+
- Angular CLI v21:
  ```bash
  npm install -g @angular/cli@21
  ```


Installation
=

1. Clone the repo:
   ```bash
   git clone <url>
   cd <project-folder>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

Running the App
=

1. Run the server
   ```bash
   ng serve
   ```
2. Open browser and navigate to http://localhost:4200
